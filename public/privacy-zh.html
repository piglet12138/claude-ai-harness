English Version

Lite Claude UI 是一个运行于 claude.yaoyuheng2001.me 的个人 AI 文档工作空间。本政策说明您在使用本应用时，我们会处理哪些信息。

我们处理的信息

  • 用于访问本应用的登录信息。
  • 您提交的消息、提示词、上传的文件、图片以及生成的文档。
  • 您授权 Google 文档上传后产生的 Google OAuth 令牌。

Google 数据

当您连接 Google 时，本应用会请求 Google Drive 文件权限，以便将您选择上传的文档创建为 Google Docs 文件。本应用仅将 Google 访问权限用于创建这些文件并打开生成的文档链接，不会出售 Google 用户数据或将其用于广告。

存储

对话和文档数据可能存储在您的浏览器本地。Google OAuth 令牌存储在服务器上，以支持授权后的文档上传。API 密钥和 OAuth 密钥不会在前端代码中暴露。

使用分析与反馈

为了提升服务质量，我们会在您与 AI 助手交互时收集匿名化的使用遥测数据。包括：

  • 交互元数据 — 时间戳、Token 用量、响应延迟、对话中发起的工具调用及可用工具列表。
  • 消息摘要 — 您输入内容的截断摘要（最多 2,000 字符）会被记录，用于诊断失败模式。完整对话历史不会被记录到遥测日志中。
  • 反馈信号 — 当您点击某条回复上的"赞"或"踩"按钮时，反馈类型及助手消息的简短摘要会被记录。

这些数据以追加写入的日志文件形式存储在服务器上，仅用于内部服务改进 — 例如识别常见失败模式、构建评估测试集。这些数据不会与第三方共享、不会用于广告、不会被出售。

您可以联系站点运营者申请删除您的遥测和反馈数据。

第三方服务

本应用可能会将您的提示词、上传内容和图片发送给配置的 AI API 提供商以生成回复。如果启用了网页搜索，搜索查询可能会被发送到配置的搜索服务提供商。

数据共享

数据仅与运行本应用所需的服务提供商共享，如配置的 AI API 提供商、用于文档上传的 Google API，以及可选的搜索 API。数据不会被出售。

联系我们

如有隐私问题或数据删除请求，请通过 yaoyuheng2001.me 的域名所有者联系方式联系站点运营者。
