隐私政策

English Version

Claude AI Harness 是一个运行于 claude.yaoyuheng2001.me 的个人 AI 文档工作空间。本政策说明您在使用本应用时，我们会处理哪些信息。

我们收集什么

当您使用本应用时，以下数据会存储在服务器上：

类别存储的数据表 / 存储位置
账户邮箱地址、哈希密码、注册日期users
对话您发送的所有消息及 Claude 的全部回复（完整内容）threads、messages
文档生成或上传的文档及其版本历史documents
长期记忆模型跨对话保存的持久化笔记（您可以随时查看、编辑或清空）user_memory
遥测工具调用记录、您输入的前 ~2000 字符（消息摘要）、Token 用量、响应延迟、所用模型telemetry
反馈您对回复点击的"赞"或"踩"评分ratings
Bug 反馈您通过"Bug"按钮提交的文字描述和截图，报告中附带您的邮箱bug-reports.json、bug-images/

你的权利

您拥有以下数据主体权利，可直接在应用内行使：

  • 查看 / 编辑长期记忆 —— 点击顶栏的 记忆 按钮，可查看、编辑或清空您的持久化记忆。
  • 导出全部数据（数据可携权） —— 点击顶栏的 我的数据，再点击 📥 导出全部数据 (JSON)，即可下载包含账户信息、所有对话、文档、记忆、遥测、反馈和 Bug 报告的完整 JSON 文件。
  • 彻底删除账号和所有数据（被遗忘权） —— 点击顶栏的 我的数据，再点击 🗑 彻底删除我的账号和所有数据。系统会要求您输入邮箱确认，随后进入 5 秒倒计时后方可最终删除。删除完成后，您的账号、所有对话、文档、记忆、遥测、评分和 Bug 报告将被永久清除，注销登录。
  • 删除单个对话 —— 点击侧边栏中任意对话旁边的删除按钮。

管理员可见范围

站点管理员可以通过后台面板（/admin）查看所有数据，包括：

  • 所有用户账号（邮箱、注册日期、角色）
  • 所有用户的全部对话和消息
  • 所有 Bug 反馈和附带截图
  • 汇总的遥测和反馈数据

这种管理员访问权限用于调试、滥用排查和服务改进。管理员与站点运营者（yaoyuheng2001.me）为同一人。您的对话内容不会与任何第三方共享，除非为运营服务所必需（例如将提示词发送到 Anthropic API 以生成回复）。

Google 数据

当您连接 Google 时，本应用会请求 Google Drive 文件权限，以便将您选择上传的文档创建为 Google Docs 文件。本应用仅将 Google 访问权限用于创建这些文件并打开生成的文档链接，不会出售 Google 用户数据或将其用于广告。

存储

对话和文档数据可能存储在您的浏览器本地。Google OAuth 令牌存储在服务器上，以支持授权后的文档上传。API 密钥和 OAuth 密钥不会在前端代码中暴露。

使用分析与反馈

为了提升服务质量，我们会在您与 AI 助手交互时收集使用遥测数据。包括：

  • 交互元数据 —— 时间戳、Token 用量、响应延迟、对话中发起的工具调用及可用工具列表。
  • 消息摘要 —— 您输入内容的截断摘要（最多 2,000 字符）会被记录，用于诊断失败模式。完整对话历史不会被记录到遥测日志中。
  • 反馈信号 —— 当您点击某条回复上的"赞"或"踩"按钮时，反馈类型及助手消息的简短摘要会被记录。

这些数据存储在服务器上，仅用于内部服务改进——例如识别常见失败模式、构建评估测试集。这些数据不会与第三方共享、不会用于广告、不会被出售。

第三方服务

本应用可能会将您的提示词、上传内容和图片发送给配置的 AI API 提供商以生成回复。如果启用了网页搜索，搜索查询可能会被发送到配置的搜索服务提供商。

数据共享

数据仅与运行本应用所需的服务提供商共享，如配置的 AI API 提供商、用于文档上传的 Google API，以及可选的搜索 API。数据不会被出售。
